# Security Policy

This repository contains documentation, prompt text, and media references; it does not ship production application code.

Report security-sensitive issues through the repository's [Security page](https://github.com/Evolink-AI/awesome-seedance-2.0-prompts/security) and use private vulnerability reporting when available. Do not open a public issue for leaked credentials, malicious links, compromised media, or vulnerabilities affecting the related EvoLink API/skill surface.

For ordinary broken links, attribution corrections, or prompt content issues, use the public issue templates.
